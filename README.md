# Pique

A fitness & wellness habit tracker built from a Figma prototype.

**Stack:** React 18 · Vite · Tailwind CSS · React Router · localStorage

## Run locally

```bash
npm install
npm run dev
```

Open the printed URL (default `http://localhost:5173`).

## Build

```bash
npm run build   # outputs to dist/
```

## Screens

Splash · Sign In · Sign Up · Track Preference · Share Preference · Plan Selection · Dashboard · Exercise Tracker · Exercise Diary · Top 5 Workouts · Trackers · Water · Steps · Joy List · Goals · Stress · Rest Days · Fitness Bingo · Chat · Profile

## Notes

- Mobile-first; designed at iPhone width.
- Auth is mocked — any email/password signs you in.
- Designs in [`figma-exports/`](./figma-exports).
